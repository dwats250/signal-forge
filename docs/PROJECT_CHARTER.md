# Project Charter

## Mission

Turn market data into durable, reviewable trading judgment.

## Core Principles

- Preserve domain-specific thinking
- Separate calculation from interpretation
- Protect nonlinear options logic
- Execution consumes, does not reinterpret
- Learning is auditable, not automatic

## System Domains

- Macro Regime
- Geopolitical Premium
- Market Quality
- Options Behavior
- Execution
- Learning

## Initial Objective

Create a clean architectural foundation for a multi-agent trading intelligence system without introducing implementation coupling, UI assumptions, or execution logic.
