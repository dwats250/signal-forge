# Signal Forge Dashboard Skill

## Goal
Compress complex market data into fast, actionable insight.

## Core Philosophy

- Density > whitespace
- Clarity > aesthetics
- Speed > beauty
- Signal > decoration

## Layout Priorities

### Top (Above the Fold)
- Market posture
- Key drivers
- Immediate risks

### Primary Section
- Top signals (A tier only)
- Must be instantly visible

### Secondary Section
- Watchlist setups
- Compact and scannable

### Bottom Section
- Rejected / notes
- Low visual priority

## Design Rules

### 1. No wasted space
- Avoid large padding
- Avoid oversized fonts
- Fit maximum useful data on screen

### 2. Tabular + card hybrid
- Use rows for comparability
- Use cards for focus items

### 3. Visual encoding
- Color = direction (green/red)
- Badges = confidence / grade
- Position = importance

### 4. Scan speed
User should answer in 3 seconds:
- what's happening?
- what matters?
- what can I trade?

## Output Expectations

When invoked:
1. Provide compact layout
2. Use minimal text
3. Emphasize signal visibility
4. Avoid decorative elements
